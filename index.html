Structure of a C++ Program

A C++ program typically consists of the following components:

1. Preprocessor Directives

These include libraries like #include <iostream> which are necessary for input and output operations.

2. Namespace Declaration

using namespace std; allows you to use standard C++ functions without prefixing them with std::.

3. Main Function

The main() function is the entry point of any C++ program. It contains the actual code to be executed.

4. Statements and Expressions

C++ statements perform actions like variable declarations, input/output operations, and calculations.

5. Return Statement

The return 0; statement indicates that the program executed successfully.
